Detailed Description

STRAWMAN: manage the .log file of the watch. Log files have a fixed line size as defined in the LDMS_WATCH environment variable options. This enables strict resource management and fast log collection by samplers.

Function Documentation

Create a log file for arbitrary messages, typically used for to the run input file lines.

The log will be sized and watched to the extent defined by the options in the LDMS_WATCH environment variable.

Add a complete line to the log.

Line format is up to the library, but it will end with "%s\n" and $s will apply to line. At present, the line format is "$line:$epochtime:$line\n".
